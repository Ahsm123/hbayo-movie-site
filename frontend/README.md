# React + Vite

## Features

- Browse movies
- View detailed information for each movie
- Search functionality with instant feedback
- Trailer
- Wishlist
- Similar movies
- Sidescroll + dropdown on mobile

## Tech Stack

- Vite
- React
- Tailwind CSS
- TMDB REST API
- React Router
